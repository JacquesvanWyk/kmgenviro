# Phase 3: Public Frontend - Raw Idea

Build the complete public-facing website for KMG Environmental Solutions using Livewire Volt, Flux UI components, and Tailwind CSS 4.

## Core Requirements

- Homepage with hero section showcasing KMG's expertise
- Service pages (listing + individual service details)
- Team member profiles
- Project portfolio with filtering by sector/province
- Blog listing and individual posts
- Training course catalog with booking capability
- Equipment rental catalog with quote request
- Contact page
- Fully responsive design (mobile-first)
- Modern, professional design matching KMG brand (#1e7e34 green)
- SEO optimization for all pages
- Fast performance with minimal JavaScript

## Tech Stack
- Laravel 12 with Livewire 3 + Volt
- Flux UI components (free edition)
- Tailwind CSS 4
- Inter font family
- Alpine.js (included with Livewire)

## Design Principles
- Clean, professional environmental consultancy aesthetic
- Green branding (#1e7e34)
- Mobile-first responsive design
- Fast loading, optimized images
- Clear calls-to-action
- Easy navigation
- Accessibility compliant
